# Random Ideas - 个人编程项目集合

**中文** | [English](README.md)

这是一个全面的个人编程实验、学习项目和实现代码的集合，涵盖多种编程语言和技术领域。本仓库代表了一段从基础算法到高级机器学习应用的编程学习之旅。

## 📋 概述

本仓库是一个多样化的编程项目游乐场，包含：
- **Python** 应用程序，涵盖机器学习、计算机视觉、算法和自动化脚本
- **Swift** iOS 应用程序，包括计算器和记忆卡片游戏
- **Objective-C** 学习材料和 iOS 开发项目
- **C/C++** 算法实现和多线程实验
- **Go** 语言面试准备和学习材料
- **JavaScript/Electron** 桌面应用程序项目
- **深度学习**模型，使用 TensorFlow、Keras 和 PyTorch
- **计算机视觉**项目，使用 OpenCV
- **固件分析**工具，用于路由器固件逆向工程
- **Web 开发**项目，包括个人网站

## 📚 快速开始

### 环境准备

#### Python 项目
```bash
# 安装 Python 3.8+
python --version

# 安装常用依赖
pip install numpy pandas tensorflow keras opencv-python torch
```

#### Swift/Objective-C 项目
- macOS 系统，安装 Xcode 14.0 或更高版本
- iOS SDK 15.0+
- CocoaPods（用于依赖管理）

#### C/C++ 项目
```bash
# 安装 GCC/Clang
gcc --version
g++ --version

# 安装构建工具
sudo apt-get install build-essential  # Linux
brew install gcc                       # macOS
```

#### Go 项目
```bash
# 安装 Go 1.16+
go version
```

#### Electron 项目
```bash
# 安装 Node.js 和 npm
node --version
npm --version

# 进入 ElectronLatex 并安装依赖
cd ElectronLatex
npm install
```

### 运行项目

#### Python 项目
```bash
# 进入特定的 Python 项目
cd Python相关/Python\ Examples/

# 运行脚本
python calculator.py
```

#### Swift 项目
```bash
# 在 Xcode 中打开
cd Swift学习/Calculator
open Calculator.xcodeproj

# 从 Xcode 构建并运行
```

#### Electron 项目
```bash
cd ElectronLatex
npm start
```

#### C/C++ 项目
```bash
# 编译并运行
cd MultiThreadSort
g++ -std=c++11 -pthread MultiThreadSort.cpp -o multithread_sort
./multithread_sort
```

#### Go 项目
```bash
cd Golang-Learning/day1-simple-program
go run simple.go
```

## 🔧 常见问题解答

### Q: 如何选择学习路径？
A: 建议按以下顺序：
1. 从 `Python相关/Python学习入门/` 开始学习 Python 基础
2. 进入 `Python相关/算法图解/` 学习算法
3. 探索 `Tensorflow与Keras相关/` 学习深度学习
4. 如果对 iOS 感兴趣，可以学习 `Swift学习/` 目录

### Q: 项目可以直接用于生产环境吗？
A: 不建议。这些项目主要用于学习和实验，未经过充分的生产环境测试。

### Q: 如何报告问题？
A: 请在 GitHub Issues 中创建新的 issue，并说明：
- 具体的项目或目录
- 问题描述和重现步骤
- 您的运行环境信息

### Q: 是否支持 Windows 系统？
A: 大部分 Python 和 C/C++ 项目支持 Windows。Swift 和 Objective-C 项目需要 macOS 系统。Electron 项目支持跨平台。
